# User Guide

## Getting Started

### Installation

1. Install from Chrome Web Store or load unpacked extension
2. Pin the extension to your toolbar for easy access
3. Visit any Wikipedia page with mathematical formulas

### Quick Start

1. Click the extension icon to open settings
2. Enable auto-highlighting if desired
3. Visit a Wikipedia page with formulas
4. Hover over formulas to see tooltips
5. Click formulas to select variables

## Basic Features

### Variable Highlighting

- Hover over any formula to see available variables
- Click a formula to open the variable selector
- Check boxes next to variables to highlight them
- Use color picker to change highlight colors

### Color Management

- Click color swatch next to variable
- Use color picker for custom colors
- Choose from predefined color schemes
- Adjust opacity with slider

### Formula Navigation

- Use tooltip arrows to navigate between formulas
- Click formula to focus and highlight
- Double-click to open in full view
- Right-click for context menu options

## Advanced Features

### Section Control

- Use section tree view in popup
- Collapse/expand sections
- Enable/disable highlighting by section
- Drag to select multiple sections

### Color Presets

1. **Save Presets**

   - Set up your preferred colors
   - Click "Save Preset"
   - Name your preset
   - Access it anytime

2. **Share Presets**
   - Export your presets
   - Share JSON file
   - Import shared presets
   - Apply to formulas

### Performance Settings

- Adjust processing delay
- Set cache size
- Toggle lazy loading
- Control animation speed

## Tips & Tricks

### Keyboard Shortcuts

- `Alt + H`: Toggle highlighting
- `Alt + T`: Toggle tooltips
- `Alt + C`: Clear all highlights
- `Alt + S`: Save current preset

### Power User Features

1. **Quick Selection**

   - Shift+Click: Select range
   - Ctrl+Click: Add to selection
   - Alt+Click: Remove from selection

2. **Batch Operations**

   - Select multiple variables
   - Apply same color
   - Clear multiple highlights
   - Save as group

3. **Custom Styles**
   - Adjust highlight opacity
   - Change border style
   - Modify tooltip position
   - Set animation speed

## Troubleshooting

### Common Issues

1. **Highlighting Not Working**

   - Check if extension is enabled
   - Refresh the page
   - Clear browser cache
   - Check console for errors

2. **Slow Performance**

   - Reduce number of highlighted variables
   - Enable lazy loading
   - Increase processing delay
   - Clear cache

3. **Colors Not Saving**
   - Check storage permissions
   - Export presets as backup
   - Clear extension data
   - Reinstall extension
